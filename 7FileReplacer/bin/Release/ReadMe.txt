本软件用于批量替换Windows7 System32目录下的系统文件。
最佳替换方法：
1.关闭所有系统无关程序，杀软什么的劫持你自己看着办吧，自己用MSE是一点问题没有的，数字什么的就难说了，自己看着办吧。
2.复制你准备好的一堆要替换的文件到ReplaceFiles目录下面，如果你的系统以前没有替换过系统文件，且没有备份，强烈建议勾选软件右下角的“备份原文件”，这样会在BackUpFiles目录下备份与ReplaceFiles目录下对应的原版系统文件方便以后恢复。
3.运行程序点击替换，成功后会问你是否要重启，此时最好就重启。
4.重启后运行我写的另一个小软件7LibraryIconChanger，用里面的刷新图标和缩略图缓存功能刷一下缓存，资源管理器重启后应该就会正常显示替换的图标了。图标缓存不要再重启前刷新，会出错。
5.还原就是把BackUpFiles目录下备份的文件放到ReplaceFiles目录下再替换一次。
